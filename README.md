# MCP_JobAutomation_FineTuned

Agentic job-application pipeline. Phase 1: LangGraph-based orchestration
that tailors a resume to a job description with one focused LLM call for
genuine judgment and deterministic Python everywhere else. Phase 2 will
swap the LLM nodes for a small fine-tuned model.

See `feat/orchestration-scaffold` (open PR) for the initial implementation.
